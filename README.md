Si la imagen no está en el repositorio, copia la captura en `app/static/img/screenshot.png` (o en `docs/screenshot.png`) y luego ejecuta:

```bash
git add app/static/img/screenshot.png
git add README.md
git commit -m "Añadir captura para README"
git push
```

Si prefieres que yo suba la imagen por ti, súbela aquí en el chat o indícame la ruta al archivo local y la subiré al repositorio.
